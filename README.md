# IGC Interpreter

Aplicação Node.js para interpretar arquivos IGC e visualizar dados de voo.

## Uso

```bash
npm start
```

Acesse http://localhost:3000

Envie um arquivo IGC e visualize o JSON formatado com:

- Informações do piloto e aeronave
- Data do voo
- Quantidade de fixes (pontos GPS)
- Altitudes máxima e mínima
- Duração do voo (minutos)
- Primeiros 100 fixes do voo
- Informações de task (se disponível)
